# Security Policy

## About This Project

This is a static educational website. There are no user accounts, login systems, databases, server-side code, or analytics. The site is plain HTML/CSS/JavaScript served from GitHub Pages.

## What We Take Seriously

- **No student data collection**: do not add forms, tracking pixels, analytics, or anything that captures user behavior or identity
- **No third-party login**: do not add authentication mechanisms
- **No external script loading**: do not add CDN-hosted scripts from third parties
- **Content safety for children**: lesson content must be appropriate for ages 8–12

## Reporting a Security Concern

Open a GitHub issue in this repository describing the concern. For concerns that should be handled privately, use GitHub's private vulnerability reporting feature (Security › Advisories › Report a vulnerability).

## Contributors: What Not to Add

- Analytics or telemetry of any kind
- Forms that submit data to external services
- Login or authentication flows
- External CDN scripts loaded at runtime
- Tracking pixels or beacon URLs
