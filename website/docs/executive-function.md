# Executive Function Skills for Media Projects and Investigations

Media literacy often asks kids to compare sources, track claims, organize evidence, revise messages, and finish media projects. Executive function makes that work visible and manageable.

This page is the local doorway into the Literacy for Kids [Executive Function Toolkit](https://www.literacy-for-kids.com/docs/toolkits/executive-function), connected to the media analysis this curriculum builds.

## A few core ideas

- **Investigations need organized questions and sources.** A tracker keeps evidence from getting lost.
- **A media project gets easier with a checklist.** Visible steps lower the barrier to starting.
- **Prioritizing helps when a message has too many ideas.** Lead with the main idea.
- **Revision needs feedback and a next step.** "What should Version 2 change?"

## When this shows up

- When researching a claim
- When comparing multiple sources
- When planning a media message
- When revising a project after feedback
- When a group project has roles
- When too many tabs or posts compete for attention

## Tools that help

- **Source tracker** — claim, source, evidence, question.
- **Claim/evidence checklist** — what to gather before deciding.
- **Message checklist** — the steps to build a clear message.
- **Priority sort** — main idea first.
- **Restart bookmark** — for pausing research mid-investigation.

:::tip Executive Function Moment
A media investigation gets easier when the sources are visible. Keep a simple tracker: claim, source, evidence, and question. That way your brain does not have to hold every tab at once.
:::

:::caution Everyday task skills, not a diagnosis
These are everyday planning and task-management tools, not therapy, medical advice, disability evaluation, ADHD treatment, or a replacement for trusted adults, educators, caregivers, or qualified support. Kids should not be treated as lazy or broken when starting, focusing, organizing, or finishing is hard. If tasks feel consistently overwhelming, inaccessible, unsafe, or severely distressing, involve a trusted adult and appropriate support.
:::
