# Collaboration Skills for Media Projects

Media literacy often involves group research, source checking, scripts, visuals, feedback, and publishing choices. Collaboration helps kids coordinate claims, evidence, roles, message goals, and revisions.

This page is the local doorway into the Literacy for Kids [Collaboration Toolkit](https://www.literacy-for-kids.com/docs/toolkits/collaboration), connected to the media analysis this curriculum builds.

## A few core ideas

- **Media projects need a shared message goal.** "What do we want the audience to understand?"
- **Source checking can be split into roles.** Different people can track different claims and sources.
- **Disagreement can improve a message.** "That part might confuse the audience" makes the work better.
- **Feedback and revision work better when roles are clear.** Everyone knows what they are revising.

## When this shows up

- When planning a group media message
- When checking sources together
- When deciding what evidence belongs
- When revising a poster, video, slideshow, or script
- When presenting or publishing as a group

## Tools that help

- **Shared message goal** — one sentence the whole group can say.
- **Roles** — source checker, evidence tracker, designer, presenter.
- **Check-in questions** — what is done, stuck, or needs help?
- **Disagree and build frame** — concerns pointed at the message, not the person.
- **Finish together credit** — every role named at the end.

:::tip Collaboration Moment
A media group needs one shared message goal. Before making slides, posters, or scripts, ask: "What do we want the audience to understand?" That goal helps every role aim at the same target.
:::

:::caution Everyday collaboration skills, not unsafe conflict management
These are everyday collaboration tools, not therapy, bullying intervention, conflict mediation, legal advice, or a replacement for trusted adults, educators, caregivers, or qualified support. Kids should not be expected to manage unsafe, coercive, bullying, discriminatory, or threatening group situations alone. If group work feels unsafe, targeted, or too big, involve a trusted adult.
:::
